Lab 5: Functions and Graphics

------------------------
DESCRIPTION

In this lab, a series of macros and procedures are defined to allow you to draw different types of circles on the MARS bitmap.

------------------------
FILES

-
Lab5.asm

This file includes the assembly code of the lab.

-
lab5_s20_test.asm

This file includes assembly code of a test case for the lab.

------------------------
INSTRUCTIONS

This program is intended to be run using the MIPS Assembler and Runtime Simulator (MARS). The bitmap display should be opened on a test case file. The bitmap specifications intended to be used are:
Unit Width in Pixels: 1
Unit Height in Pixels: 1
Display Width in Pixels: 128
Display Height in Pixels: 128
Base address for display: 0xffff0000 (memory map)
