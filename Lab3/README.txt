Lab 3: ASCII-risks (Asterisks)

------------------------
DESCRIPTION

In this lab, the user can input an integer value greater than 0, causing numbers and stars to be printed out in an inverted pyramid pattern.

------------------------
FILES

-
Lab3.asm

This file includes the assembly code of the lab.

------------------------
INSTRUCTIONS

This program is intended to be run using the MIPS Assembler and Runtime Simulator (MARS). Assemble the program, run, then input a valid integer number (greater than 0).
