Lab 4: Sorting Integers

------------------------
DESCRIPTION

In this lab, the user inputs hexadecimal numbers as program arguments then the program prints out these arguments, their integer values (unsorted), and their sorted integer values.

------------------------
FILES

-
Diagram.pdf

This file includes a block diagram of the program design. The application draw.io was used to draft the diagram and generate the pdf.

-
Lab4.asm

This file includes the assembly code of the lab.

------------------------
INSTRUCTIONS

This program is intended to be run using the MIPS Assembler and Runtime Simulator (MARS). Enter the test case as a program argument.
